<h1>:game_die:Dev Sorteio</h1>

<p>Aplicação web para sorteio de números aleatórios dentro de um intervalo definido pelo usuário.</p>
<p>Projeto desenvolvido durante meus estudos no DevClub, com o objetivo de aprimorar a lógica de programação, aplicar boas práticas de desenvolvimento e trabalhar design responsivo com foco em usabilidade.</p>
